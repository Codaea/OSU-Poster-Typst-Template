# OSU Poster Typst Template

Template(s) for posters written in typst

I am not affiliated with Oregon State University. This template is a personal project and is not endorsed by OSU.

## How to use

Install [typst](https://github.com/typst/typst) or use the [web version](https://typst.app/). 

Then, copy the contents of `poster.typ` into a new typst file and edit the content as needed. You can also customize the styles defined at the top of the file.

OSU Fonts are not included, and can be found at [communications.oregonstate.edu](https://communications.oregonstate.edu/brand-guide/typography)
